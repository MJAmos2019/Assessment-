    Project Idea

    Overview The project idea is a program that could also be a smart phone app used by doctors and patients to monitor their health. The application or program would gather data on patients who are already in the hospital. Data that is significant and can show changes in the patients condition, such as blood levels, heart rate, muscle activity and other vital measurements. This program would use artificial intelligence to continually gather data on each patient so that when there is a significant enough change doctors can be notified immediately to take the proper course of action, so that human error can be minimised. Motivation There was a study in the United States carried out by John Hopkins which found that over 250,000 people each year in the US die from medical errors. This makes it the third leading cause of death in the country. Some other studies state the number exceed 400,000. That number of people is far to significant to be ignored, if there can be a solution to decrease or even eradicate that number it would seem more than welcomed. If artificial intelligence can decrease doctor error and pick up things that humans may not be able to notice straight away then thousand of lives could be save through the development of such technology. Description The aim of this machine is to consistently measure vital signs in order to detect any potential changes in a patient that otherwise would have been missed. It aim also it to provide quick and accurate detection of any issues with the patients health as it arises.The application would work in combination with other existing technologies so it can be effective. Since it would present a challenge to build up new machines and technologies that would measure vital signs the option that seems best would be to utilised what is currently in hospitals around the world. Machines that already monitor and measure vital parameters would transfer data to the application. These parameters include but are not limited to the following:Body temperature, Pulse rate,Breathing rate (respiration), Blood pressure and Intrinsic measurements of the blood and muscle activity. All these and more will be used to determine medical interventions of the patients. The data collected can be processed and analysed by artificial intelligence using well designed algorithmic calculations. Over time as the AI receives more data it will become increasingly more accurate as it would have drawn data from a wide range of patients over time. This will ensure that the aim of this project will be reached.
